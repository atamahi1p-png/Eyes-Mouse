# Eyes Mouse

## AI Eye Tracking Mouse Controller

Eyes Mouse is an Android application that allows users to control a computer mouse using eye movements with the help of AI-powered eye tracking technology.

### Features
- Real-time Eye Tracking
- Cursor Control
- Blink Click Detection
- Hands-Free Navigation
- User Friendly Interface
